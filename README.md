# 倾听动物测试 · Listening Type Indicator (LTI)

> Which **listening animal** are you? A bilingual, interactive assessment that turns *how you listen* into one of six animal archetypes — and helps you listen better.
>
> 你是哪种**倾听动物**？一个中英双语的交互测评，把"你怎么听"映射成六种动物原型，并帮你听得更好。

## English

**Listening Type Indicator (LTI)** is an interactive emotional-health micro-service built around a simple idea: *being truly heard is one of the rarest experiences in modern life.* Answer 12 everyday scenarios to discover which of six **listening animals** you are — then go further: see how you and a friend listen *together*, and practice becoming a better listener over 7 days.

### ✨ Features
- **12-question scenario test** with weighted, multi-dimensional scoring → one of **6 listening animals** (Whale · Owl · Dolphin · Tiger · Bear · Butterfly).
- **Rich result profile** — strengths, blind spots, growth tips, and the six-level listening science.
- **Listening Duo (关系合盘)** — pick you + a friend for a bespoke two-person dynamic report (21 hand-written combinations); share via a `?pair=` link.
- **7-Day Practice (练习营)** — a personalized micro-practice journey targeting your blind spot, progress saved locally.
- **Listening Lab (倾听学堂)** — a companion knowledge page (empathy, OARS methods, myths, self-care…).
- **Shareable result card** (Canvas + QR), fully **bilingual**.

### 🧠 Grounding
Carl Rogers' empathic listening · Zenger & Folkman's six-level listening scale (HBR, 2016) · Motivational Interviewing (OARS) · Listening Styles Profile.

### 🛠 Tech
Pure static **HTML / CSS / vanilla JS** — no build, no framework, no backend. State persists in `localStorage`. Open `index.html` to run.

## 中文

**倾听动物测试（Listening Type Indicator, LTI）** 是一个交互式的情感健康小服务，核心是一个朴素的洞察：*"被真正听见"，是现代人最稀缺的体验之一。* 回答 12 个日常情境，发现你是六种**倾听动物**中的哪一种；再往前一步——看看你和朋友"在一起怎么听"，并用 7 天练习成为更好的倾听者。

### ✨ 功能
- **12 题情境测评**，加权多维计分 → 六种倾听动物（鲸 · 枭 · 豚 · 虎 · 熊 · 蝶）；
- **丰富结果页** —— 优势、盲点、成长建议、六级倾听科学；
- **关系合盘** —— 选你 + 朋友，生成专属双人互动报告（21 种组合，全部手写）；可用 `?pair=` 链接分享；
- **7 天练习营** —— 针对你盲点的每日微练习，进度本地保存；
- **倾听学堂** —— 配套知识页（共情、OARS 方法、误区、自我关怀…）；
- **可分享结果卡**（Canvas + 二维码），**中英双语**。

### 🧠 理论支撑
Carl Rogers 共情倾听 · Zenger & Folkman 六级倾听量表（HBR, 2016）· 动机式访谈（OARS）· 倾听风格模型。

### 🛠 技术
纯静态 **HTML / CSS / 原生 JS** —— 无构建、无框架、无后端，状态存于 `localStorage`，打开 `index.html` 即可运行。
